Conseils d’activation

Exactitude

Copiez-collez le code pour éviter les erreurs de saisie.

Conservation

Gardez l’e-mail contenant le code comme preuve d’achat.

Assistance

En cas d’échec, contactez le support avec le code et l’heure.

Étapes

  1. Collez votre code et vérifiez votre e-mail (optionnel).
  2. Validez pour déclencher l’activation côté serveur.
  3. Recevez un e-mail de confirmation si un e-mail est fourni.

FAQ

Mon code est-il réutilisable ?

Non, chaque code est à usage unique et consommé à l’activation.

Que faire si le code est invalide ?

Vérifiez l’orthographe, sinon contactez le support avec le reçu.
